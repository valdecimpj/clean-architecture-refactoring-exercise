# Clean Architecture Refactoring Exercise
## Acquiring a release
[Download a release here.](https://github.com/valdecimpj/clean-architecture-refactoring-exercise/releases)
## Windows
Execute Web.exe, a web browser should open with the Swagger UI.
## Linux and MacOS
In the extracted folder, run `chmod +x ./Web`.  
Run `./Web`.  
If there's a GUI, a browser should open with Swagger UI  
\* Was not tested in MacOS
